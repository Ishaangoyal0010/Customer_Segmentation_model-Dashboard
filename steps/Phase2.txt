15. Dockerize the backend (write Dockerfile inside app/)

# Dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY . .
RUN pip install --no-cache-dir -r requirements.txt
EXPOSE 8000
CMD ["uvicorn", "app.api:app", "--host", "0.0.0.0", "--port", "8000"]

   create .dockerignore and add:
   .git/
   venv/
   __pycache__/
   mlruns/
   .gitignore
   notebooks/

16. Build and run docker container

>docker build -t segmentation-backend .
>docker run -d -p 8000:8000 segmentation-backend

   open http://localhost:8000/docs to verify ✔️

17. Push backend image to Docker Hub

>docker login
