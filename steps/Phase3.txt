20. Dockerfile for frontend (inside app/)

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 8501
CMD ["streamlit", "run", "streamlit_app.py", "--server.port=8501", "--server.address=0.0.0.0"]

24. Docker Compose to run both backend and frontend together
    create docker-compose.yml at project root

version: '3'
services:
  backend:
    build: ./app
    ports:
      - "8000:8000"
    volumes:
      - ./artifacts:/app/artifacts
  frontend:
    build: ./app
    ports:
      - "8501:8501"
    depends_on:
      - backend
    environment:
      - API_URL=http://backend:8000

25. Run everything together

>docker-compose up

   backend  → http://localhost:8000/docs
   frontend → http://localhost:8501
   test full flow: upload CSV on streamlit → hits FastAPI → returns segments ✔️
