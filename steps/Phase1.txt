VIRTUAL ENVIRONMENT SETUP ✔️

6. Download dataset from Kaggle
   search "Online Retail UCI" on Kaggle
   download CSV → rename to ecommerce.csv
   put it inside data/raw/

7. Run EDA notebook
   open notebook/eda.ipynb in Jupyter
   run all cells top to bottom

   what it does:
   - loads data/raw/ecommerce.csv
   - removes missing CustomerID rows
   - removes cancelled orders (InvoiceNo starts with C)
   - removes negative quantity and price rows
   - creates TotalSpend = Quantity * UnitPrice
   - builds RFM features (Recency, Frequency, Monetary)
   - caps outliers at 99th percentile
   - saves cleaned data to data/processed/cleaned.csv

   check data/processed/cleaned.csv is created ✔️

8. Run feature engineering

>cd src
>python feature_engineering.py

   what it does:
   - loads data/processed/cleaned.csv
   - scales RFM features using StandardScaler
   - saves scaler to artifacts/scaler.pkl

   check artifacts/scaler.pkl is created ✔️

9. Train models

>python train.py

   what it does:
   - trains 4 models: KMeans, Agglomerative, GaussianMixture, DBSCAN
   - logs all runs to MLflow (mlflow.db)
   - picks best model by silhouette score
   - assigns segment labels: Champions, Loyal Customers, New Customers, At Risk
   - saves PCA plot to artifacts/pca_clusters.png
   - saves best_model.pkl, scaler.pkl, pca.pkl, segmented_customers.csv

   check artifacts/ folder has all 4 files ✔️

10. Check MLflow experiment tracking

>mlflow ui --backend-store-uri sqlite:///mlflow.db

   open http://localhost:5000
   you should see 4 runs with silhouette, davies_bouldin, calinski scores

   EXPERIMENT TRACKING ✔️
